# Singularity+
Singularity+ is a fork of another KSP mod (Singularity) which aimed at adding ray-traced black hole shaders, gravitational lensing, wormholes, etc.
This fork incorporates jcyuan06s PR on adding doppler effects within Singularitys source, this is technically a seperate version of Singularity rather then an addon. (you either use Singularity or Singularity+ so they do not conflict)
  
Singularitys original Repository: https://github.com/LGhassen/Singularity <br />Original Documentation: https://github.com/LGhassen/Singularity/wiki <br />the PR that this fork of Singularity incorporates: https://github.com/LGhassen/Singularity/pull/8
